# 🔢 Binary Number System

The **Binary Number System** uses only two digits:

- `0`
- `1`

It is the foundation of modern computer systems and digital electronics.

## 📋 Decimal to Binary Conversion Table

| Decimal | Binary |
|---|---|
| 0 | 0000 |
| 1 | 0001 |
| 2 | 0010 |
| 3 | 0011 |
| 4 | 0100 |
| 5 | 0101 |
| 6 | 0110 |
| 7 | 0111 |
| 8 | 1000 |
| 9 | 1001 |

# ⚡ Operators in Programming

Operators are symbols used to perform operations on variables and values.

## ➕ 1. Arithmetic Operators

Used for mathematical calculations.

### 🔹 Binary Operators

| Operator | Description |
|---|---|
| `+` | Addition |
| `-` | Subtraction |
| `*` | Multiplication |
| `/` | Division |
| `%` | Modulus |

### 🔹 Unary Operators

| Operator | Description |
|---|---|
| `++` | Increment |
| `--` | Decrement |

---

## 🔄 2. Increment & Decrement Operators

| Type | Example | Description |
|---|---|---|
| Pre Increment | `++a` | Value increases before use |
| Post Increment | `a++` | Value increases after use |
| Pre Decrement | `--a` | Value decreases before use |
| Post Decrement | `a--` | Value decreases after use |

---

## 📊 3. Relational Operators

Used to compare two values.

| Operator | Meaning |
|---|---|
| `==` | Equal to |
| `!=` | Not Equal to |
| `>` | Greater than |
| `<` | Less than |
| `>=` | Greater than or Equal to |
| `<=` | Less than or Equal to |

---
